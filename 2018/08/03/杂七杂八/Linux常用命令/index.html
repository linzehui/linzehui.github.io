Linux常用命令

发表于 2018-08-03

记录自己常用的命令。

1️⃣ls：显式当前目录下的文件和目录
-a 包括隐藏文件

2️⃣top：动态查看系统运行状态
-u 指定用户名
-p 指定进程

3️⃣kill 杀死程序
-l 信息编号。当l=9时，无条件终止，其他信号可能忽略

4️⃣ps 报告当前系统的进程状态
-a 所有
-p 指定程序
-u 指定用户

5️⃣htop 比top更优，交互更好，同时可以直观看到资源占用情况
基本命令与top一致

6️⃣cd 到达指定地址

7️⃣nvidia-smi 查看显卡状态

8️⃣vim 使用vim打开指定文件
:q 退出
:wq 写入并退出
:q! 退出并忽略所有更改
:e! 放弃修改并打开原来的文件
:u 撤销
:n 跳转到行号 如， :100
:set nu 显示行号
:set nonu 隐藏行号
i 插入模式

9️⃣echo 用于打印指定的字符串
